# ADR-999: Legacy Payments (Deprecated)

- **Status:** Deprecated (2024-10-01)
- **Context:** Ранее существовал модуль `payment-service`, который взаимодействовал с внешним PSP.
  Код был удалён в 2026 году, но документ остаётся для исторической ссылки.
- **Risk:** Любые упоминания платежей в текущих артефактах должны рассматриваться как технический долг.
- **Action:** При следующем цикле исследовательского аудита удалить документ либо перенести в архив.
